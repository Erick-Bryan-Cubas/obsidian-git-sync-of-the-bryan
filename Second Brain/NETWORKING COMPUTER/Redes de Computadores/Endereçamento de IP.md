Endereçammento de [[IP]]

Um IP é constituído de 4 [[octetos]] e 32 [[bits]]

192.168.0.1
192 = 11000000
168 = 10101000
0 = 00000000
1 = 00000001

128 | 64 | 32 | 16 | 8 | 4 | 2 | 1 |
---- | --| -- | -- | -- | -- | -- | -- |
1   | 1 |  0 |  0  |  0  |  0  |  0  |  0  | 


Da esquerda para direita soma-se os números. Assim obtemos os bits de 192.
